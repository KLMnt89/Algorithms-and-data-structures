# Algorithms and Data Structures

A collection of algorithm implementations and solutions completed during the
**Algorithms and Data Structures** course at the Faculty of Computer
Science and Engineering — Skopje (2024/2025, Winter Semester).

## 📁 Repository Structure
```
├── DynamicProgramming/   # Dynamic programming problems & solutions
├── Graphs/               # Graph traversal and algorithms
├── Hash/                 # Hash table problems & solutions
├── StackArray/           # Stack and Array based solutions
└── Threes/               # Tree data structures & algorithms
```

## 🛠️ Topics Covered

- **Dynamic Programming** — memoization, tabulation, optimization problems
- **Graphs** — traversal (BFS/DFS), shortest path, connectivity
- **Hash Tables** — hashing, collision handling, lookup problems
- **Stacks & Arrays** — stack-based algorithms, array manipulation
- **Trees** — binary trees, traversals, tree-based problems

## 🚀 How to Run

All solutions are implemented in **Java**. To run any file:
```bash
javac FileName.java
java FileName
```

Or open the project in **IntelliJ IDEA** / **Eclipse** and run directly.
